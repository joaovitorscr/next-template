# Next.js template

## 🧱 Stack
* Next.js
* TailwindCSS
* Eslint
* Prettier

## ⚠️ Dependencies

Node >=20.11

## 🔌 How to use it?

### ⌨️ Execute the code below

```
npx create-next-app --example https://github.com/joaovitorscr/next-template <APP-NAME> --use-<Your package manager(npm, pnpm, yarn)>
```
